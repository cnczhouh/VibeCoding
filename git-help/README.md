## Git 简易助手

这是一个给 Git 新手使用的小工具，把常见 Git 操作做成中文按钮。

### 最简单用法

你不需要先懂 Git。日常只按这个顺序：

1. 打开工具，点 `选择文件夹`
2. 点 `看看当前情况`
3. 看蓝色的 `下一步建议`
4. 改完文件后，点 `保存这次修改`
5. 需要上传/下载时，点 `和远程同步`

第一次保存失败时，大概率是还没设置身份，点 `设置名字和邮箱`。

如果你选择的是项目里的某个子文件夹，工具只会查看、保存、备份和撤销这个子文件夹里的修改。
工具还会拦截常见的运行缓存、依赖目录和数据库文件，避免误把这些内容保存到 Git。

如果看到本地已经 `ahead`，表示电脑上有还没上传的保存记录。确认内容没问题后再点同步。

### 怎么启动

在 Windows 上双击：

```text
start_git_helper.cmd
```

或者在当前文件夹运行：

```text
python git_easy.py
```

### 能做什么

- 查看当前项目有没有未保存修改
- 一键保存当前版本
- 拉取别人最新修改
- 上传自己的版本
- 创建和切换分支
- 连接远程仓库地址
- 设置 Git 保存记录里的名字和邮箱
- 备份未保存修改
- 撤销已被 Git 管理的文件修改

撤销操作默认会保留新建文件，避免误删还没被 Git 记录的新内容。

### 为什么 Git 管理目录可能更靠上

Git 通常会管理一整个项目文件夹。如果你选择的是项目里的子文件夹，例如 `E:\mylife\git-help`，但真正的 Git 仓库在 `E:\mylife`，工具会保留你选择的位置，同时单独显示 `Git 管理目录：E:\mylife`。
