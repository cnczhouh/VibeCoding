# Prompt Studio

一个本地生图提示词工作台，内置 YouMind GPT Image 2 提示词库。

## 启动

```powershell
npm install
Copy-Item .env.example .env
# 在 .env 里填 OPENAI_API_KEY
npm run server
npm run dev
```

前端默认地址：`http://127.0.0.1:5180`

后端默认地址：`http://127.0.0.1:8787`
